zm-next — detect on 720p substream, overlay on 4K main stream (WebVTT metadata)

—

The <video> plays the original 4K H.264 with no re-encode. The <track kind="metadata"> carries the detect_onnx plugin's Event.DETECTION JSON — but detection ran on the cheap 720p substream. Each cue records the detection resolution, so the canvas scales the boxes from 720p onto the 4K display for free. cuechange fires and the canvas draws them; the checkbox toggles the layer like a subtitle.
